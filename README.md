🌸 Memory Scent Reciper

📌 Project Overview
This project uses Generative AI to convert human memory descriptions into structured scent formulations.

🚀 Features
- Converts memory to fragrance notes
- Uses GPT-2 / DistilGPT2 model
- Generates:
  - Top notes
  - Middle notes
  - Base notes
  - Chemical molecules

🧠 Tech Stack
- Python
- Transformers (Hugging Face)
- NLP
- Google Colab

📂 Dataset
- Custom dataset + perfume dataset (1000+ entries)

▶️ How to Run
1. Install requirements
2. Load saved model
3. Run UI cell

💡 Example
Input:
> Bakery with vanilla smell

Output:
Top: sweet, vanilla  
Middle: creamy  
Base: caramel  
Molecules: Vanillin

🎯 Future Scope
- Web app deployment
